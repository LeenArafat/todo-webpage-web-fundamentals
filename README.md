# Web Fundamentals ToDo Webpage

This repository contains a simple **ToDo webpage** project completed for the **Web Fundamentals** course.  
The main goal of the project was to implement interactive functionality using **JavaScript** on pre-provided HTML and CSS files.

## Project Overview
- Built a ToDo list webpage where users can add and remove tasks.
- Implemented core functionality using **JavaScript**.

## Key Contributions
- Developed all interactive JavaScript features for managing tasks.
- Ensured proper DOM manipulation and event handling.

## Tools & Technologies
- HTML (structure of the webpage)
- CSS (styling and layout)
- JavaScript (interactive functionality)
